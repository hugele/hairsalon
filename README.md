# アプリ名

Brown Bunny

# アプリケーション概要

架空のヘアサロン「Brown Bunny」のホームページ

# URL

https://hugele.github.io/hairsalon/index.html

# 利用方法

１　トップ画面に「Menu」「News」「Contact」へ遷移するボタンがある。

2 「Contact」ページでは予約とスタイリストの指名ができる
